# Morphema – Demo concettuale

Questa applicazione è una DEMO concettuale che rappresenta un singolo **incarico autonomo occasionale a risultato**.

Non rappresenta:
- lavoro subordinato
- collaborazione continuativa o para-subordinata
- turnazione o ricorrenze

Ogni incarico in demo è:
- evento unico
- non continuativo
- a risultato
- svolto in autonomia dal professionista occasionale
- su incarico di un committente

## Flusso demo (1 storia)

1) Committente: crea un incarico (bozza)  
2) Committente: preautorizza → pubblica  
3) Professionista: vede 1 incarico e lo accetta (candidatura)  
4) Committente: accetta la candidatura
